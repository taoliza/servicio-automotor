# Instrucciones para ejecutar el proyecto

## Backend (Spring Boot)

1. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/taoliza/servicio-automotor
   cd servicio-automotor
   ```

2. **Compilar y ejecutar los tests**:
   Desde la raíz del proyecto, ejecuta:
   ```bash
   mvn clean install
   mvn test
   ```

3. **Levantar la API**:
   Si los tests pasan correctamente, ejecuta:
   ```bash
   mvn spring-boot:run
   ```
   Esto levantará el servidor backend en `http://localhost:8080`.

## Frontend (React)

1. **Navegar a la carpeta del frontend**:
   ```bash
   cd frontend-servicio-automotor
   ```

2. **Instalar dependencias**:
   Ejecuta el siguiente comando para instalar todas las dependencias necesarias:
   ```bash
   npm install
   ```

3. **Iniciar la aplicación**:
   Una vez instaladas las dependencias, inicia la aplicación con:
   ```bash
   npm start
   ```
   Esto levantará el servidor de desarrollo de React en `http://localhost:3000`.
   Se abrirá un navegador con la Aplicación ya lista para usar en modo de prueba.
